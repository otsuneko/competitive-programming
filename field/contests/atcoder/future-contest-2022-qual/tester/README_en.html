Farm King X tools

We provide the contestant tools for RECRUIT Nihonbashi Half Marathon 2021 (long) problem Farm King X.

  • These programs don't guarantee the actual score of the contest.
  • Use these programs at your own risk.
  • These programs are provided as-is. We don't accept any question about them.
  • You can use these programs in your solution.

Test case generation

You can generate a test case by providing a random seed to generator.py

These commands generate a test case and write it into input.txt, using 12345 as the random seed value.

python generator.py 12345 > input.txt

or

python3 generator.py 12345 > input.txt

input_0.txt ~ input_9.txt are generated test cases with the seed 0 ~ 9 respectively.

Score calculation

You can use judge.py to calculate the score of your output.

These commands calculate the score of output.txt against the test case input.txt.

python judge.py input.txt output.txt

or

python3 judge.py input.txt output.txt
