Visualizer for 農場王X

これは何ですか？

RECRUIT 日本橋ハーフマラソン 2021〜増刊号〜 の 農場王X のビジュアライザです。

  • このビジュアライザはデスクトップ版の Google Chrome および Mozilla Firefox の最新バージョン上で動作確認を行っています。全てのブラウザ環境で動作することを保証していません。
  • このビジュアライザ上で計算された得点は、当コンテストでの得点ではありません。解答を AtCoder 上で提出する事によって採点が行われます。また、ビジュアライザ上で計算された得点は、当コンテスト上での得点を保証するものではありません。
  • このビジュアライザを使用したことで発生したトラブルには対応できません。ご了承ください。

使い方

  • index.html をブラウザで開きます。
  • 画面上部の input file と output file に、テストケースジェネレータが生成したファイルと、あなたのプログラムが出力した内容のファイルを指定します。
  • エラーがなければ、右の money にスコアが表示されます。
  • エラーがある場合は、ダイアログにエラーメッセージが表示されます。

save as image のボタンから、png 形式の画像として保存できます。SNS でシェアして SNS 映えさせましょう。
